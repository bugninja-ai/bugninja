## Available Files for Upload

You have access to the following files that can be uploaded to file input fields:

[[FILE_LIST]]

**To upload a file:**
1. Identify the file input element on the page
2. Use the `upload_file` action with the file's index number
3. The system will automatically handle the file upload using the actual file path

**Important:**
- Only use file indices listed above
- Files are pre-validated and ready for upload
- Do not attempt to specify file paths directly
- Use the `upload_file` action with both the element index and file index
